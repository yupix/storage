# 一覧

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| ファイルの有無 | Any |
| エンドポイント | /v1/folders |
| メソッド | GET |

## 概要

ユーザーが指定したフォルダーの内部ファイルを全て表示することができます。

## 必要なデータ

| キー | 値の種類 |
|----|----|
|    |    |

## 表示するデータ

| キー | 値の種類 |
|----|----|
| folder_name | String |
| folder_updated_at | datetime |
| file_name | String |
| file_size | integer |
| file_updated_at | datetime |

## 実装要件
- フォルダー内ファイル・サブフォルダーの再帰取得またはシャロー取得
- 必要コンポーネント: PostgreSQL
