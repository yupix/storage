# 削除

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| エンドポイント | /v1/forders/:id |
| メソッド | DELETE |


## 概要

フォルダーを削除します。

オプションで中にあるファイルをホームディレクトリにばらまくことも可能

## 必要なデータ



| キー | 値の種類 |
|----|----|
| folder_id | String |
| to_home | Boolean |

## 実装要件
- to_home=true 時: 子ファイルをルートディレクトリへ移動
- to_home=false 時: 子ファイルも論理削除
- 必要コンポーネント: PostgreSQL
