# プロフィール取得

## 概要
ログイン中の自分のアカウント情報を取得する。

## エンドポイント情報

| 要件 | 値 |
|----|-----|
| ログインの有無 | 要 |
| HTTPメソッド | GET |
| エンドポイント | /v1/accounts/me |

## 必要なデータ
セッションから自分のユーザーIDを特定。追加パラメータなし。

## レスポンス

| キー | 値の種類 | 説明 |
|------|---------|------|
| user_id | String | ユーザーID |
| email | String | メールアドレス |
| is_frozen | Boolean | 凍結状態 |
| created_at | datetime | 作成日時 |

## 実装要件
- Valkey でセッション → ユーザーID取得
- PostgreSQL でユーザー情報取得
