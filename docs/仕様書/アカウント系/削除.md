# 削除

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| 管理者権限の有無 | True |
| エンドポイント | /v1/accounts/me |
| メソッド | DELETE |


## 概要

ユーザーが不要になったアカウントを本人確認を行った後、データベースから論理削除する。

### 本人確認の方法

パスワードを入力させ、一致した場合のみアカウントの削除を行う。


## 必要なデータ

| キー | 値の種類 |
|----|----|
| パスワード | String |


## 表示するデータ

| キー | 値の種類 |
|----|----|
|    |    |

## 実装要件
- 論理削除フラグの更新（deleted_at 等）
- セッション無効化（Valkey）
- 必要コンポーネント: PostgreSQL、Valkey
