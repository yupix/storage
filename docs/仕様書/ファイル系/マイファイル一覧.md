# マイファイル一覧

## 概要
自分がアップロードしたファイルの一覧を取得する。ゴミ箱内は含まない。

## エンドポイント情報

| 要件 | 値 |
|----|-----|
| ログインの有無 | 要 |
| HTTPメソッド | GET |
| エンドポイント | /v1/files/mine |

## クエリパラメータ

| キー | 値の種類 | 必須 | デフォルト | 説明 |
|------|---------|------|-----------|------|
| folder_id | String | False | - | 指定フォルダー内のみ（省略時はルート） |
| page | Integer | False | 1 | ページ番号 |
| limit | Integer | False | 50 | 件数（最大: 100） |

## レスポンス
| キー | 説明 |
|------|------|
| files | Array: ファイルオブジェクト |
| total | Integer: 全件数 |
| page | Integer: 現在ページ番号 |
| limit | Integer: 1ページ件数 |

## 実装要件
PostgreSQL owner_id = 自分 でページネーション付き取得
