# 更新

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| エンドポイント | /v1/files/:id |
| メソッド | PATCH |


## 概要


ファイルの情報を更新します。

* ファイルが存在するフォルダーを変更する
* ファイル名を変更する

など


## 必要なデータ



| キー | 値の種類 | 必須 |
|----|----|----|
| folder_id | String | False |
| filename | String | False |

## 実装要件
- DBのメタデータ更新
- 必要コンポーネント: PostgreSQL
