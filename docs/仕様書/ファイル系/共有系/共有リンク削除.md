# 共有リンク削除

## 概要
発行済みの共有リンクを無効化・削除する。

## エンドポイント情報

| 要件 | 値 |
|----|-----|
| ログインの有無 | 要 |
| HTTPメソッド | DELETE |
| エンドポイント | /v1/files/:id/links/:link_id |

## 必要なデータ

| キー | 値の種類 | 必須 | 説明 |
|------|---------|------|------|
| link_id | String | True | 削除対象のリンクID |

## 実装要件
- PostgreSQL: 共有リンク情報を削除（論理削除または物理削除）
