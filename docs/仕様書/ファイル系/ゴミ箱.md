# ゴミ箱

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| エンドポイント | /v1/files/trash |
| メソッド | GET |


### 概要

ホームのフォルダから削除したファイルを削除したうえで復元可能状態に置き換わる

今のところ一度削除した復元可能ファイルは30日経過で完全に削除させる方向で計画中

サービスによっては一度削除してから復元可能期間を設定できるものもあるらしい。

## 実装要件
- 削除フラグ付きファイルの一覧取得
- 30日経過後の物理削除（RustFS + DBレコード削除）
- 必要コンポーネント: PostgreSQL、RustFS、定期ジョブ（cron）
