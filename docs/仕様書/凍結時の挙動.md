# 凍結時の挙動

## ログイン中

どのような状態でもログインが解除され、メインメニューに戻されます

## ログアウト状態

凍結されたアカウントにログインを実行すると"このアカウントは凍結されています。"と表示されログインに失敗します。

## 保存ファイルの取り扱い

凍結されたアカウントの共有中のファイルを共有されているユーザーがアクセス(ダウンロード、閲覧、編集)することはできません。\nアクセスしようとすると"このファイルの所有者は凍結されています。"と表示されます

## 凍結時の連絡

凍結されたアカウントのメールアドレス宛に凍結された理由とアカウントが凍結されたことを連絡します。
