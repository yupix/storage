# メールサービス

## 役割

凍結通知メールの送信（[アカウント凍結API](../../api/auth/suspend.md)で使用）。

## 実装方式の選択肢

| 方式 | 例 |
|------|-----|
| SMTP直接 | 自前SMTPサーバー・Gmailリレー |
| メール配信API | SendGrid / Resend / AWS SES |

## 送信内容

| 項目 | 内容 |
|------|------|
| 宛先 | 凍結アカウントのメール |
| 件名 | `[HyperDrive] アカウントが凍結されました` |
| 本文 | 凍結理由・日時 |

## 非機能要件

- 凍結操作と**非同期送信**（失敗しても API レスポンスはブロックしない）
- 送信失敗時はログ記録
