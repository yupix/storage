# 凍結解除

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| 凍結状態 | True |
| 管理者権限の有無 | True |
| エンドポイント | /v1/accounts/:id/unfreeze |
| メソッド | PATCH |

## 概要

データベース上で凍結になっているアカウントを再利用できるようにする制御。


## 必要なデータ

| キー | 値の種類 |
|----|----|
| ユーザーID | string |
| 日付 | datetime |

## 実装要件
- 凍結フラグ解除（DB）
- 必要コンポーネント: PostgreSQL
