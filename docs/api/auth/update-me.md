# プロフィール更新

| 要件 | 値 |
|----|----|
| ログインの有無 | 要 |
| エンドポイント | /v1/users/me |
| メソッド | PATCH |


## 概要

ログイン中の自分のユーザー名・メールアドレスを更新する。送信されたフィールドのみ更新する（部分更新）。


## 必要なデータ

| キー | 値の種類 | 説明 |
|----|----|----|
| username | String（任意・3文字以上） | 新しいユーザー名。省略時は変更しない |
| email | String（任意・メール形式） | 新しいメールアドレス。省略時は変更しない |


## レスポンス

更新後のユーザー情報（`UserResponse`）を返す。

| ステータス | 内容 |
|----|----|
| 200 | 更新後のユーザー情報 |
| 400 | バリデーションエラー |
| 401 | 未認証 |
| 409 | ユーザー名またはメールアドレスが既に使用されている |


## 実装メモ（2026-07-15）

- Rust API (`handlers/users.rs::update_me`) で実装。`username` / `email` は unique 制約があるため、更新前に重複を事前チェックして 409 を返す。
- OpenAPI スキーマ（`apps/web/src/api/schema.d.ts`）に反映済み。フロントは設定ページ（`/settings`）から利用。


## 実装要件
- Valkey でセッション → ユーザーID取得
- PostgreSQL でユーザー情報更新（`updated_at` も更新）
- 必要コンポーネント: PostgreSQL、Valkey
