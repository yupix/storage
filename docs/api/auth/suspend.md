# 凍結

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| 管理者権限の有無 | True |
| エンドポイント | /v1/accounts/:id/freeze |
| メソッド | PATCH |


## 概要

データベース上でフラグ制御を用いてアプリやサービスの**アカウントが一時的または永久的に利用できなくなる状態**にする。


## 必要なデータ

| キー | 値の種類 |
|----|----|
| ユーザーID | String |
| 日付 | datetime |
| 凍結理由 | String |


\

## 実装要件
- 凍結フラグの更新（DB）
- セッション強制無効化（Valkey）
- 凍結通知メール送信
- 必要コンポーネント: PostgreSQL、Valkey、メール送信サービス
