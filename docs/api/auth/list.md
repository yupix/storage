# 一覧

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| 管理者権限の有無 | True |
| エンドポイント | /v1/accounts |
| メソッド | GET |

## 概要

データベース上にあるアカウント全てをアカウント状態にかからわず表示します

## 表示するデータ

| キー | 値の種類 |
|----|----|
| ユーザーID | String |
| メールアドレス | String |
| 凍結状態 | String |

## 実装要件
- 管理者権限チェック（セッション検証）
- 必要コンポーネント: PostgreSQL、Valkey
