# ファイル名検索

> **統合済み**: ファイル名検索は OCR 検索と統合されています。`/v1/search` エンドポイントを参照してください。

| 要件 | 値 |
|----|----|
| ログインの有無 | True |
| エンドポイント | /v1/search |
| メソッド | GET |

## 概要

ファイル名に含まれる文字列で検索できる機能。OCR テキスト検索と統合された `/v1/search` エンドポイントで処理される。

### 最大検索可能文字数

255文字まで(ファイル名の最大文字数)

## 必要なデータ

| キー | 値の種類 | 必須 | 説明 |
|------|---------|------|------|
| q | String | True | 検索キーワード（最大255文字）|

## 実装要件
- PostgreSQL の `filename ILIKE` 検索（大文字小文字無視）
- 必要コンポーネント: PostgreSQL
