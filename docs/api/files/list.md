# 一覧

| 要件 | 値 |
|----|----|
| ログインの有無 | any |
| エンドポイント | /v1/files |
| メソッド | GET |


### 概要

共有されているファイルを見ることができる。open形式で共有されている場合ログインは不要だが、secret形式で共通されている場合はログインが必要となる。

## 必要な情報

| キー | 値の種類 |
|----|----|
|    |    |

## 表示する情報

| キー | 値の種類 |
|----|----|
| ファイル名 | String |
| データサイズ | Integer |
| 更新日時 | datetime |
| 送信者ID | String |

## 実装要件
- 共有形式（open/secret）によるアクセス制御ロジック
- 必要コンポーネント: PostgreSQL
